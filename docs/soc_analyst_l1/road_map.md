If you're starting from scratch and want to become a **Security Operations Center (SOC) Analyst**, here's a practical roadmap that takes you from fundamentals to job-ready skills.

## Phase 1: Build IT Fundamentals (2–4 weeks)

Learn the basics of:

* Computer hardware and operating systems
* Networking fundamentals

  * TCP/IP
  * DNS
  * HTTP/HTTPS
  * DHCP
  * VPN
* Windows administration
* Linux fundamentals
* Command line (PowerShell and Bash)

**Resources**

* Cisco Packet Tracer
* [Microsoft Learn](https://learn.microsoft.com)
* [Linux Journey](https://linuxjourney.com)

---

## Phase 2: Networking (3–4 weeks)

A SOC analyst spends a lot of time analyzing network traffic.

Topics:

* OSI Model
* TCP vs UDP
* Ports and protocols
* Routing
* Firewalls
* VPNs
* NAT
* Packet capture

Practice tools:

* Wireshark
* tcpdump

Projects:

* Capture your own traffic.
* Analyze HTTP requests.
* Detect DNS queries.
* Understand TCP handshakes.

---

## Phase 3: Operating Systems (3 weeks)

### Windows

Learn:

* Event Viewer
* Active Directory basics
* Registry
* Services
* Scheduled Tasks
* PowerShell

### Linux

Learn:

* File permissions
* SSH
* Cron jobs
* Processes
* Logs
* Bash scripting

Practice using:

* Ubuntu>
* Windows 11

---

## Phase 4: Cybersecurity Fundamentals (4 weeks)

Study:

* CIA Triad
* Authentication
* Authorization
* Encryption
* Hashing
* MFA
* Malware types
* Phishing
* Ransomware
* Vulnerability management

Frameworks:

* MITRE ATT&CK
* OWASP
* National Institute of Standards and Technology

---

## Phase 5: SOC Fundamentals (4 weeks)

Understand:

* Incident lifecycle
* Alert triage
* False positives
* IOC vs IOA
* Threat Intelligence
* Log analysis
* SIEM
* EDR
* SOAR

Learn common log sources:

* Windows logs
* Linux logs
* Firewall logs
* DNS logs
* Proxy logs
* VPN logs

---

## Phase 6: Learn SIEM Tools (4–6 weeks)

Focus on one first.

Popular choices:

* Splunk Enterprise
* Microsoft Sentinel
* Elastic Security
* IBM QRadar

Practice:

* Writing searches
* Creating dashboards
* Building alerts
* Investigating incidents

---

## Phase 7: Detection & Threat Hunting

Learn:

* Sigma rules
* YARA basics
* IOC hunting
* Threat hunting methodology

Tools:

* Sysmon
* Velociraptor
* Kibana

---

## Phase 8: Incident Response

Study:

* Malware analysis basics
* Memory analysis
* Disk forensics
* Evidence collection
* Containment
* Eradication
* Recovery

Tools:

* Volatility
* Autopsy

---

## Phase 9: Scripting (Highly Recommended)

Learn Python:

* Reading logs
* Parsing CSV
* APIs
* Automation

Also learn:

* Bash
* PowerShell

---

## Phase 10: Practice Labs

Best platforms:

* [TryHackMe](https://tryhackme.com)
* [Hack The Box Academy](https://academy.hackthebox.com)
* [Blue Team Labs Online](https://blueteamlabs.online)
* [LetsDefend](https://letsdefend.io)

Complete blue-team and SOC-focused scenarios rather than only penetration testing labs.

---

## Phase 11: Certifications (Optional but Valuable)

Suggested order:

1. Google Cybersecurity Certificate
2. CompTIA Security+
3. Splunk Core Certified User
4. Microsoft Certified: Security Operations Analyst Associate (SC-200)
5. CompTIA CySA+

---

## Phase 12: Build a Portfolio

Include:

* SIEM dashboards
* Detection rules
* Threat hunting reports
* Malware analysis write-ups
* Incident response reports
* Home lab documentation
* Python automation scripts

Publish your work on:

* [GitHub](https://github.com)
* [LinkedIn](https://www.linkedin.com)

---

## Home Lab

Create a small SOC lab using:

* VirtualBox or VMware Workstation
* Windows Server
* Ubuntu
* Splunk Enterprise or Elastic Security
* Sysmon

Generate logs and practice detecting simulated attacks.

## Suggested 6-Month Study Plan

| Month | Focus                                                                |
| ----- | -------------------------------------------------------------------- |
| 1     | Networking + Linux + Windows fundamentals                            |
| 2     | Cybersecurity fundamentals + Wireshark + logs                        |
| 3     | SIEM (Splunk or Microsoft Sentinel) + SOC concepts                   |
| 4     | Incident response + MITRE ATT&CK + threat hunting                    |
| 5     | Python + home lab + detection engineering                            |
| 6     | Practice labs, portfolio projects, resume, and interview preparation |

By the end of this roadmap, you should be prepared for entry-level roles such as **SOC Analyst (L1)**, **Security Analyst**, **Blue Team Analyst**, or **Junior Incident Responder**. A strong portfolio with documented investigations and lab work can significantly strengthen your applications, especially if you have limited professional experience.
